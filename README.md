# Customer Segmentation using K-Means Clustering

This project implements K-Means clustering to segment retail store customers based on their purchasing behavior.

## Dataset:
Customer Segmentation Dataset from Kaggle

## Features Used:
- Age  
- Annual Income  
- Spending Score  

## Tech Stack:
- Python  
- Pandas  
- Scikit-learn  
- Matplotlib  

## Project Tasks:
1. Implemented K-Means clustering  
2. Used Elbow Method to determine optimal clusters  
3. Visualized clusters using 2D plots  
4. Implemented 3D clustering visualization
